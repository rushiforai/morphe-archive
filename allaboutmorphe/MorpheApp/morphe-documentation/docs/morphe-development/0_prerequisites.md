# 💼 Prerequisites

Certain requirements are needed to be able to develop with Morphe.

## 🤝 Requirements

- A Java/ Kotlin IDE such as [Android Studio](https://developer.android.com/studio) or [IntelliJ IDEA](https://www.jetbrains.com/idea/) with the Android plugin. 
- Java Development Kit 21 or newer (e.g. [Azul Zulu](https://www.azul.com/downloads/?version=java-21-lts&package=jre#zulu))
- Basic knowledge of Git and Gradle
