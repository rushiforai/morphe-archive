## 🚀 Usage

### Add as a source in Morphe Manager

1. Install [Morphe Manager](https://morphe.software) on your Android device
2. Open **Sources** → **Add Source**
3. Enter:
   ```
   https://github.com/MoonShadowKeeper/Telegram-patchesMorphe
   ```
4. Select **Telegram 12.8.3** APK and apply the patches you want

> **Note:** Download the correct APK format (`XAPK`/`APKM`) for your architecture (`arm64` for modern phones).

## ❓ FAQ

#### What APK should I use?
Download **Telegram 12.8.3** (`org.telegram.messenger.web`) in XAPK or APKM format, arm64 architecture.

#### Why can't I log in with Google?
Patched apps are re-signed, which breaks Google Play Services authentication. This is a known limitation.

#### The patch doesn't work?
Open an [issue](https://github.com/MoonShadowKeeper/Telegram-patchesMorphe/issues) with your app version and error details.
