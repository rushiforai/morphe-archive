# Contributing

This is a personal patches repository. Issues and pull requests are welcome if you spot a bug or have an improvement to the screenshot restriction patch.

## Reporting a bug

Open an [issue](https://github.com/Pa-kon/morphe-screenshot-patches/issues) describing what went wrong and which Instagram version you're on.

## Submitting a fix

1. Fork the repository and create a branch from `main`
2. Make your changes
3. Open a pull request describing what you changed and why
