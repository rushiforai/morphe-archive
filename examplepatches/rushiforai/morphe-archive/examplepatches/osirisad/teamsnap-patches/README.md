## 🚀 Get started

1. Install Morphe Manager and switch it to advanced/expert mode
2. [Click here to add this patch source in Morphe](morphe://addSource?url=https://raw.githubusercontent.com/osirisad/teamsnap-patches/main/patches-bundle.json)
   - Or manually add this URL in Morphe Manager: `https://raw.githubusercontent.com/osirisad/teamsnap-patches/main/patches-bundle.json`
3. In Morphe Manager, find TeamSnap
4. Select "Disable ads" patch and patch the app
5. Install and enjoy ad-free TeamSnap!
