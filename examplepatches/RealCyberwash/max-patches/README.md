# Max Patches (Android)

Patches to reduce government control

## 📦 Included Patches

* **Disable VPN Detection:** Disables VPN detection mechanisms. The IP checker used in analytics is restricted to use only the Russian Yandex server.

## 🚀 How to use these patches

Click the link below to automatically add these patches to Morphe Manager:
[Add to Morphe](https://morphe.software/add-source?github=RealCyberwash/max-patches)

Or manually add this repository URL as a patch source in Morphe Manager:
`https://github.com/RealCyberwash/max-patches`

### 🛠️ Building

To build these patches manually, you can follow the [Morphe documentation](https://github.com/MorpheApp/morphe-documentation).
